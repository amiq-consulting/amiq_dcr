*Description:
              amiq_dcr VIP is an universal verification component (UVC) modeling the Device Control Register Bus (DCR) protocol

              amiq_dcr package is build on top of uagt package.
              uagt is an Universal Agent, build in SystemVerilog, following UVM metodology.
              uagt sources can be downloaded from https://github.com/amiq-consulting/uagt

*Running the example:

              You need to set system variable UAGT_HOME to the location of uagt sources.
              You need to run the script ${AMIQ_DCR_DIR}/demo.sh

              For more options of demo.sh you can run: ${AMIQ_DCR_DIR}/demo.sh -help
